# 📈 Do All in go!

> **"수천 개의 AI 에이전트가 살아 움직이는 시장에서 직접 배우는 실전 투자 교육 플랫폼"**

## 🚀 프로젝트 개요
단순한 주식 매매 툴이 아닌, 최신 AI 논문 방법론을 활용하여 복잡한 시장의 흐름과 투자자의 심리를 '주린이(입문자)'가 직접 체험하며 배울 수 있는 가상 시뮬레이션 서비스입니다.

## 📚 핵심 연구 기반 (Key Research)
이 프로젝트는 아래 두 가지 핵심 논문의 방법론을 하이브리드로 결합하여 구현됩니다.
1. **ASFM (Agent-based Simulated Financial Market):** 실제 오더 매칭 시스템 및 에이전트 기반 금융 시장 시뮬레이션.
2. **AgentSociety:** 대규모 LLM 에이전트의 사회적 상호작용 및 욕구/감정 모델링.

## 🛠 기술 스택 (Tech Stack)
- **Frontend:** React
- **Backend:** FastAPI, Ray (Distributed Engine)
- **AI/Cloud:** Azure OpenAI, Azure AI Search (RAG), Azure AI Speech (TTS)
- **Data:** Azure Database for PostgreSQL, Azure Event Grid (MQTT)
